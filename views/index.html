Documentation de l'API

Bienvenue dans la documentation de l'API. Cette API fournit des endpoints pour gérer les pilotes, les circuits, les chronos et les utilisateurs.

Retrouvez le code source du projet sur GitHub : https://github.com/votre-utilisateur/votre-projet

Endpoints disponibles

Endpoint Description Méthodes HTTP
/pilotes Récupère la liste des pilotes GET
/pilotes/{id_pilote} Récupère un pilote spécifique GET
/pilotes Crée un nouveau pilote POST
/pilotes/{id_pilote} Supprime un pilote DELETE
/pilotes/{id_pilote} Met à jour un pilote PUT
/circuits Récupère la liste des circuits GET
/circuits/{id_circuit} Récupère un circuit spécifique GET
/circuits Crée un nouveau circuit POST
/circuits/{id_circuit} Supprime un circuit DELETE
/circuits/{id_circuit} Met à jour un circuit PUT
/chronos Récupère la liste des chronos GET
/chronos/{id_chrono} Récupère un chrono spécifique GET
/chronos Crée un nouveau chrono POST
/chronos/{id_chrono} Supprime un chrono DELETE
/chronos/{id_chrono} Met à jour un chrono PUT
/official-circuits Récupère la liste des circuits officiels (API externe) GET
/register Inscrit un nouvel utilisateur POST
/login Authentifie un utilisateur POST
